## 使用流程

### 用户兑换商品流程
```
1. 用户浏览商品详情页
2. 点击"使用XX能量兑换"按钮
3. 弹出确认对话框，显示商品信息和所需能量
4. 用户点击"确定兑换"
5. 系统检查登录状态：
   - 已登录：跳转到订单创建页面
   - 未登录：提示"需要登录"，点击"去登录"跳转到登录页面
6. 用户选择登录方式：
   - 邮箱/用户名登录：输入用户名和密码
   - 微信一键登录：授权获取微信信息
7. 登录成功后返回商品详情页
8. 再次点击兑换按钮，跳转到订单创建页面
```

### 用户注册流程
```
1. 进入登录页面
2. 切换到"注册"标签
3. 填写注册信息：
   - 用户名（必填，至少3个字符）
   - 昵称（必填）
   - 邮箱（选填）
   - 密码（必填，至少6个字符）
   - 确认密码（必填）
4. 点击"注册"按钮
5. 注册成功，自动登录
6. 返回上一页面
```

### 微信登录流程
```
1. 进入登录页面
2. 点击"微信一键登录"按钮
3. 授权获取微信用户信息
4. 系统自动创建/登录账号
5. 新用户赠送100能量
6. 登录成功，返回上一页面
```

## 注意事项

1. **数据库迁移**: 执行数据库迁移脚本前请备份数据库
2. **微信配置**: 微信小程序需要在微信公众平台配置服务器域名
3. **HTTPS**: 微信小程序要求后端API必须使用HTTPS
4. **Token存储**: 前端使用 `uni.setStorageSync` 存储Token
5. **密码安全**: 后端使用 `password_hash()` 加密密码
