# 商品卡密资源绑定 - 两级选择实现指南

## 功能说明

商品添加/编辑页面的卡密资源绑定功能采用**两级选择**方式：

1. **第一级**：选择卡密分类（账号密码类、下载连接类、教程类）
2. **第二级**：在选定的分类下搜索并选择具体的卡密内容

## 后端 API（已完成）

### 获取卡密资源列表
```
GET /admin/products/card-resources?module_type=account&keyword=xxx
```

**请求参数**：
- `module_type` (可选): 卡密分类
  - `account` - 账号密码类
  - `download` - 下载连接类
  - `tutorial` - 教程类
- `keyword` (可选): 搜索关键词（搜索标题、用户名、URL、内容、备注）
- `page` (可选): 页码，默认 1
- `limit` (可选): 每页数量，默认 50

**响应示例**：
```json
{
  "code": 0,
  "msg": "success",
  "data": {
    "list": [
      {
        "id": 1,
        "module_type": "account",
        "title": "Roblox 账号资源",
        "username": "user123",
        "status": 1,
        "sort": 0
      }
    ],
    "pagination": {
      "page": 1,
      "limit": 50,
      "total": 10
    }
  }
}
```

### JavaScript 实现

```javascript
// 在 openProductModal 函数中添加以下代码

// 卡密资源状态
const cardResourceState = {
    id: isEdit ? Number(product.card_resource_id || 0) : 0,
    name: isEdit ? String(product.card_resource_name || '') : '',
    moduleType: isEdit ? String(product.card_resource_module_type || '') : '',
};

// 获取DOM元素
const cardResourceTypeSelect = document.getElementById('productCardResourceType');
const cardResourceSearchInput = document.getElementById('productCardResourceSearch');
const cardResourceIdInput = document.getElementById('productCardResourceId');
const cardResourceCurrentDiv = document.getElementById('productCardResourceCurrent');
const cardResourceResultsDiv = document.getElementById('productCardResourceResults');
const searchCardResourceBtn = document.getElementById('searchCardResourceBtn');

// 如果是编辑模式且已绑定，显示当前绑定信息
if (isEdit && cardResourceState.id > 0) {
    cardResourceTypeSelect.value = cardResourceState.moduleType;
    cardResourceSearchInput.disabled = false;
    searchCardResourceBtn.disabled = false;
    cardResourceCurrentDiv.innerHTML = '<div class="tag tag-green">已绑定：' 
        + escapeHtml(cardResourceState.name) 
        + ' <span class="tag tag-blue">' + escapeHtml(getModuleTypeName(cardResourceState.moduleType)) + '</span>'
        + ' (ID: ' + cardResourceState.id + ')'
        + '</div>'
        + '<button type="button" class="btn btn-danger btn-small" id="unbindCardResourceBtn">解除绑定</button>';
}

// 第一级：选择卡密分类
cardResourceTypeSelect.addEventListener('change', function () {
    const selectedType = this.value;
    
    if (selectedType === '') {
        // 未选择分类，禁用搜索
        cardResourceSearchInput.disabled = true;
        cardResourceSearchInput.value = '';
        searchCardResourceBtn.disabled = true;
        cardResourceResultsDiv.innerHTML = '';
    } else {
        // 选择了分类，启用搜索
        cardResourceSearchInput.disabled = false;
        searchCardResourceBtn.disabled = false;
        cardResourceResultsDiv.innerHTML = '<div class="muted">请输入关键词搜索，或点击搜索按钮查看全部</div>';
    }
});

// 第二级：搜索卡密资源
searchCardResourceBtn.addEventListener('click', async function () {
    const moduleType = cardResourceTypeSelect.value;
    const keyword = cardResourceSearchInput.value.trim();
    
    if (moduleType === '') {
        alert('请先选择卡密分类');
        return;
    }
    
    try {
        const data = await api('/admin/products/card-resources', {
            query: {
                module_type: moduleType,
                keyword: keyword
            }
        });
        
        if (!data.list || data.list.length === 0) {
            cardResourceResultsDiv.innerHTML = '<div class="empty">未找到匹配的卡密资源</div>';
            return;
        }
        
        // 渲染搜索结果
        cardResourceResultsDiv.innerHTML = '<div class="product-card-resource-list">'
            + data.list.map(function (item) {
                return '<div class="product-card-resource-item">'
                    + '<div class="product-card-resource-info">'
                    + '<strong>' + escapeHtml(item.title) + '</strong>'
                    + '<span class="tag tag-blue">' + escapeHtml(getModuleTypeName(item.module_type)) + '</span>'
                    + (item.username ? '<span class="tag tag-gray">账号: ' + escapeHtml(item.username) + '</span>' : '')
                    + (item.url ? '<span class="tag tag-gray">链接: ' + escapeHtml(item.url.substring(0, 30)) + '...</span>' : '')
                    + '<span class="tag tag-gray">ID: ' + item.id + '</span>'
                    + '</div>'
                    + '<button type="button" class="btn btn-small" data-select-card-resource="' + item.id + '" data-card-resource-name="' + escapeHtmlAttr(item.title) + '" data-card-resource-type="' + escapeHtmlAttr(item.module_type) + '">选择</button>'
                    + '</div>';
            }).join('')
            + '</div>';
        
        // 绑定选择按钮事件
        document.querySelectorAll('[data-select-card-resource]').forEach(function (btn) {
            btn.addEventListener('click', function () {
                const resourceId = this.dataset.selectCardResource;
                const resourceName = this.dataset.cardResourceName;
                const resourceType = this.dataset.cardResourceType;
                
                cardResourceState.id = Number(resourceId);
                cardResourceState.name = resourceName;
                cardResourceState.moduleType = resourceType;
                cardResourceIdInput.value = resourceId;
                
                cardResourceCurrentDiv.innerHTML = '<div class="tag tag-green">已绑定：' 
                    + escapeHtml(resourceName) 
                    + ' <span class="tag tag-blue">' + escapeHtml(getModuleTypeName(resourceType)) + '</span>'
                    + ' (ID: ' + resourceId + ')'
                    + '</div>'
                    + '<button type="button" class="btn btn-danger btn-small" id="unbindCardResourceBtn">解除绑定</button>';
                
                cardResourceResultsDiv.innerHTML = '';
                cardResourceSearchInput.value = '';
                
                // 绑定解除按钮事件
                document.getElementById('unbindCardResourceBtn').addEventListener('click', function () {
                    if (!confirm('确定要解除卡密资源绑定吗？')) {
                        return;
                    }
                    
                    cardResourceState.id = 0;
                    cardResourceState.name = '';
                    cardResourceState.moduleType = '';
                    cardResourceIdInput.value = '0';
                    cardResourceCurrentDiv.innerHTML = '<div class="muted">未绑定卡密资源</div>';
                    cardResourceResultsDiv.innerHTML = '';
                    cardResourceTypeSelect.value = '';
                    cardResourceSearchInput.value = '';
                    cardResourceSearchInput.disabled = true;
                    searchCardResourceBtn.disabled = true;
                });
            });
        });
    } catch (error) {
        alert('搜索失败：' + error.message);
    }
});

// 支持回车搜索
cardResourceSearchInput.addEventListener('keydown', function (event) {
    if (event.key === 'Enter') {
        event.preventDefault();
        searchCardResourceBtn.click();
    }
});

// 辅助函数：获取卡密分类名称
function getModuleTypeName(moduleType) {
    const typeMap = {
        'account': '账号密码类',
        'download': '下载连接类',
        'tutorial': '教程类'
    };
    return typeMap[moduleType] || moduleType;
}
```

## 使用流程

### 创建商品并绑定卡密

1. 点击"新增商品"按钮
2. 填写商品基本信息
3. 在"卡密资源绑定"区域：
   - 选择卡密分类（账号密码类/下载连接类/教程类）
   - 输入搜索关键词（可选）
   - 点击"搜索"按钮
   - 从搜索结果中点击"选择"按钮
4. 确认绑定状态显示正确
5. 点击"创建"按钮保存商品

### 编辑商品修改绑定

1. 点击商品列表中的"编辑"按钮
2. 查看当前绑定状态
3. 可以：
   - 点击"解除绑定"按钮解除当前绑定
   - 重新选择卡密分类和卡密资源
4. 点击"保存"按钮更新商品

### 搜索功能

- **按标题搜索**: 输入卡密资源的标题关键词
- **按内容搜索**: 输入账号、URL、内容等关键词
- **查看全部**: 选择分类后直接点击搜索，不输入关键词

## 注意事项

1. **卡密创建时不绑定商品**: 卡密资源创建时不需要也不能绑定商品，只在商品管理页面进行绑定
2. **必须先选择分类**: 搜索前必须先选择卡密分类
3. **绑定是可选的**: 商品可以不绑定卡密资源
4. **一对一关系**: 一个商品只能绑定一个卡密资源
5. **搜索范围**: 搜索会在标题、用户名、URL、内容、备注等字段中查找

## 后端修改说明

### Product 控制器
- `getCardResources()` 方法已支持 `module_type` 参数
- 搜索范围扩展到标题、用户名、URL、内容、备注

### CardResource 控制器
- 移除了创建卡密时必须绑定商品的限制
- 卡密可以独立创建，不需要关联商品
