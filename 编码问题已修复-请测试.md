### 常见问题

**Q: 页面还是空白**
A: 确认是否清除了浏览器缓存，尝试使用 Ctrl+F5 强制刷新

**Q: 看不到卡密资源绑定区域**
A: 
1. 检查浏览器控制台是否有 JavaScript 错误
2. 确认 `card-resource-binding.css` 文件已加载
3. 检查 Network 标签，确认 `card-admin.js` 文件是最新的

**Q: 点击搜索按钮没反应**
A: 
1. 检查是否先选择了卡密分类
2. 查看浏览器控制台是否有 API 错误
3. 确认后端接口 `/admin/products/card-resources` 正常工作

## 数据库迁移

如果还没有执行数据库迁移，请执行：

```bash
php database/add_card_resource_binding.php
```

或者直接在数据库中执行 SQL：

```sql
ALTER TABLE `products` 
ADD COLUMN `card_resource_id` BIGINT UNSIGNED NOT NULL DEFAULT 0 COMMENT '关联的卡密资源ID' 
AFTER `kind_category_id`;

ALTER TABLE `products` 
ADD INDEX `idx_card_resource_id` (`card_resource_id`);
```
