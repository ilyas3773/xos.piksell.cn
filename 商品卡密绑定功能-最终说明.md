# 商品卡密资源绑定功能 - 最终实现说明

## 功能概述

为商品管理模块添加了卡密资源绑定功能，采用**两级选择**方式：

### 第一级：选择卡密分类
- 账号密码类 (account)
- 下载连接类 (download)
- 教程类 (tutorial)

### 第二级：搜索并选择具体卡密
- 支持按标题、内容、账号、URL等关键词搜索
- 显示卡密详细信息
- 点击选择按钮进行绑定

## 重要变更

### ✅ 卡密创建时不绑定商品
- 卡密资源创建时**不需要**也**不能**绑定商品
- 只在商品管理页面进行绑定
- 已移除 CardResource 控制器中"必须绑定商品"的限制

### ✅ 两级选择机制
- 第一步：选择卡密分类（账号密码类/下载连接类/教程类）
- 第二步：在选定分类下搜索并选择具体卡密

### ✅ 强大的搜索功能
- 搜索范围：标题、用户名、URL、内容、备注
- 支持模糊匹配
- 支持回车键快速搜索

## API 接口

### 获取卡密资源列表（支持两级筛选）
```
GET /admin/products/card-resources?module_type=account&keyword=xxx
```

**请求参数**:
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| module_type | string | 否 | 卡密分类：account/download/tutorial |
| keyword | string | 否 | 搜索关键词 |
| page | int | 否 | 页码，默认 1 |
| limit | int | 否 | 每页数量，默认 50 |

**响应示例**:
```json
{
  "code": 0,
  "msg": "success",
  "data": {
    "list": [
      {
        "id": 1,
        "module_type": "account",
        "title": "Roblox 账号资源",
        "username": "user123",
        "url": "",
        "status": 1
      }
    ],
    "pagination": {
      "page": 1,
      "limit": 50,
      "total": 10
    }
  }
}
```

### 解除绑定
```
POST /admin/products/:id/unbind-card-resource
```

### 创建/更新商品（支持 card_resource_id）
```
POST /admin/products
PUT /admin/products/:id

参数：
- card_resource_id: 卡密资源ID，0表示不绑定
```


### JavaScript 逻辑
1. 选择卡密分类后，启用搜索框
2. 点击搜索按钮，调用 API 获取该分类下的卡密列表
3. 显示搜索结果，每个结果有"选择"按钮
4. 点击"选择"按钮，绑定该卡密资源
5. 显示当前绑定状态和"解除绑定"按钮

详细代码请查看：`商品卡密绑定-两级选择实现指南.md`

## 使用流程

### 创建卡密资源（不绑定商品）
1. 进入卡密管理页面
2. 选择卡密类型（账号密码类/下载连接类/教程类）
3. 填写卡密信息（标题、账号、密码、URL等）
4. **不需要选择商品**
5. 保存卡密资源

### 创建商品并绑定卡密
1. 进入商品管理页面
2. 点击"新增商品"
3. 填写商品基本信息
4. 在"卡密资源绑定"区域：
   - **第一步**：选择卡密分类（账号密码类/下载连接类/教程类）
   - **第二步**：输入关键词搜索（可选）
   - **第三步**：点击"搜索"按钮
   - **第四步**：从结果中点击"选择"按钮
5. 确认绑定状态
6. 保存商品

### 编辑商品修改绑定
1. 点击商品的"编辑"按钮
2. 查看当前绑定状态
3. 可以：
   - 点击"解除绑定"解除当前绑定
   - 重新选择卡密分类和卡密资源
4. 保存商品

## 注意事项

1. **卡密创建独立**: 卡密资源创建时不需要也不能绑定商品
2. **两级选择**: 必须先选择卡密分类，才能搜索具体卡密
3. **搜索范围**: 搜索会在标题、用户名、URL、内容、备注等字段中查找
4. **绑定可选**: 商品可以不绑定卡密资源
5. **一对一关系**: 一个商品只能绑定一个卡密资源
