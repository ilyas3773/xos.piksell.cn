# 快速开始 - 商品卡密绑定功能

## 🎯 核心功能

商品可以绑定卡密资源，采用**两级选择**方式：
1. **第一级**：选择卡密分类（账号密码类/下载连接类/教程类）
2. **第二级**：搜索并选择具体的卡密内容

## 📋 重要变更

### ✅ 卡密创建时不绑定商品
- 卡密资源创建时**不需要**绑定商品
- 只在商品管理页面进行绑定
- 已移除必须绑定商品的限制

### ✅ 两级选择机制
```
选择卡密分类 → 搜索具体卡密 → 选择绑定
```

### ✅ 强大搜索功能
搜索范围：标题、用户名、URL、内容、备注

## 🔌 API 接口

### 获取卡密资源列表
```
GET /admin/products/card-resources?module_type=account&keyword=xxx
```

**参数**：
- `module_type`: account（账号密码类）/ download（下载连接类）/ tutorial（教程类）
- `keyword`: 搜索关键词（可选）

**响应**：
```json
{
  "code": 0,
  "data": {
    "list": [
      {
        "id": 1,
        "module_type": "account",
        "title": "Roblox 账号资源",
        "username": "user123"
      }
    ]
  }
}
```

### 解除绑定
```
POST /admin/products/:id/unbind-card-resource
```

### 保存商品（支持 card_resource_id）
```
POST /admin/products
PUT /admin/products/:id

参数：
- card_resource_id: 卡密资源ID（0表示不绑定）
```

## 🎨 使用流程

### 创建卡密资源
```
卡密管理 → 新增卡密 → 填写信息 → 保存
（不需要选择商品）
```

### 绑定卡密到商品
```
商品管理 → 新增/编辑商品 → 选择卡密分类 → 搜索卡密 → 选择绑定 → 保存
```

### 解除绑定
```
编辑商品 → 点击"解除绑定"按钮 → 确认 → 保存
```

## ⚠️ 注意事项

1. **卡密独立创建**: 卡密资源创建时不需要绑定商品
2. **两级选择**: 必须先选择分类，才能搜索卡密
3. **绑定可选**: 商品可以不绑定卡密资源
4. **一对一**: 一个商品只能绑定一个卡密资源
