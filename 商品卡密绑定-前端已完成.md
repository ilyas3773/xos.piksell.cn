## 功能特性

### 两级选择机制
1. **第一级**：选择卡密分类
   - 账号密码类 (account)
   - 下载连接类 (download)
   - 教程类 (tutorial)

2. **第二级**：搜索并选择具体卡密
   - 支持按标题、内容、账号、URL等搜索
   - 显示卡密详细信息
   - 点击选择按钮进行绑定

### 核心功能
- ✅ 两级选择：先选分类，再选卡密
- ✅ 强大搜索：支持多字段模糊搜索
- ✅ 卡密独立：卡密创建时不需要绑定商品
- ✅ 可选绑定：商品可以不绑定卡密资源
- ✅ 解除绑定：随时可以解除绑定关系
- ✅ 回车搜索：支持回车键快速搜索
- ✅ 响应式设计：支持移动端

## 使用流程

### 创建卡密资源（不绑定商品）
```
卡密管理 → 选择类型 → 新增卡密 → 填写信息 → 保存
（不需要选择商品）
```

### 创建商品并绑定卡密
```
商品管理 → 新增商品 → 填写基本信息 → 
选择卡密分类 → 搜索卡密 → 选择绑定 → 保存
```

### 编辑商品修改绑定
```
商品管理 → 编辑商品 → 查看当前绑定 → 
可以解除绑定或重新选择 → 保存
```

## API 接口

### 获取卡密资源列表
```
GET /admin/products/card-resources?module_type=account&keyword=xxx
```

**参数**：
- `module_type`: 卡密分类（account/download/tutorial）
- `keyword`: 搜索关键词（可选）

### 解除绑定
```
POST /admin/products/:id/unbind-card-resource
```

### 保存商品
```
POST /admin/products
PUT /admin/products/:id

参数：
- card_resource_id: 卡密资源ID（0表示不绑定）
```

## 注意事项

1. **数据库迁移必须先执行**，否则后端接口会报错
